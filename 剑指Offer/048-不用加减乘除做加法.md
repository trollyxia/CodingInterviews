# 048-不用加减乘除做加法



## 题目描述

写一个函数，求两个整数之和，要求在函数体内不得使用+、-、*、/四则运算符号。



## 思路

利用位运算。

1. 两个数异或：相当于每一位相加，而不考虑进位；

2. 两个数相与，并左移一位：相当于求得进位；

   如果这一步求得的进位值为0，则第一步中求得的结果即为最终结果。

3. 将上述两步的结果相加，

   即重复上述1,2 两步直到第2步中求得的进位值为0。



## 代码

```c++
class Solution {
public:
    int Add(int num1, int num2)
    {
        while(num2){
            int a = num1 ^ num2;
            int b = (num1 & num2) << 1;
            num1 = a;
            num2 = b;
        }
        return num1;
    }
};
```
